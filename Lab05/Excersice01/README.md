# Lab05 - Exersice 01

Write a program that accepts as arguments the pathname of an image in DICOM format and an output image pathname terminated in suffix .png , and converts the DICOM image to PNG format. Next, pass to the program each one of the two images in DICOM format provided in the data directory, DicomImage1.dcm and DicomImage2.dcm . Now that you have these two images converted to PNG, visualize them with a simple Image Viewer application.
